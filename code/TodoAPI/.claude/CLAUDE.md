# TodoAPI Project Instructions

## Using TodoAPI MCP Tools

When the user asks about **todos**, **todo items**, **tasks**, or **things to do** — including queries like:
- "List all todos"
- "Add a todo"
- "What else is left to do?"
- "Mark this as complete"
- "Get my todo list"

**Always use the TodoAPI MCP tools** to interact with the todo data, rather than making generic suggestions or checking the task list.

The TodoAPI provides tools for:
- Getting all todos
- Adding new todos
- Marking todos as complete/incomplete
- Filtering todos by completion status

Use these tools first to show the actual state of the TodoAPI before providing any other assistance.
